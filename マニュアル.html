1はじめに・起動方法

Mojicast は、マイクの音声をリアルタイムに文字起こしして配信画面に字幕として表示するアプリです。 音声認識・句読点付け・英訳まですべてお使いのPCの中だけで動きます。音声がネットに送られることはありません。

初回起動 初回だけ「WindowsによってPCが保護されました」(SmartScreen)が出ることがあります。 未署名アプリのためで、「詳細情報」→「実行」で起動できます。
ダウンロード由来のブロック(MOTW)はアプリが起動時に自動解除するので、手動の解除操作は不要です。

起動手順

1
このフォルダを丸ごとローカルディスクにコピーする（Zip 内や共有ドライブから直接起動しない）
2
Mojicast.exe をダブルクリック
3
コックピット画面が開いたら、マイクを選んで ▶開始
初回のみ モデルの読み込みに少し時間がかかります（「認識モデルをロード中...」の表示）。2回目以降は速くなります。

2コックピット（メイン画面）

起動して最初に出る画面です。左が設定、中央が字幕モニタ、右がスタイル切替と OBS 連携です。

項目説明
▶開始 / ■停止音声認識のオン・オフ
マイク入力デバイスの選択。「既定のマイク」で通常はOK
無音で確定この長さ黙ると1行が確定する。短いほどテンポが良く、長いほど1行が長くなる
途中更新間隔話している最中の「薄文字」の更新頻度
最大発話長喋り続けたとき、この秒数で強制的に確定する
単語ブースト登録単語の効きの強さ。効かない→上げる / 誤爆する→下げる
句読点をつける確定文に「、」「。」を自動で付ける
単語登録を使う単語スタジオのホットワードを使う
数字を算用数字に漢数字を算用数字へ変換（三十五→35、1万5000円 など）。 「一緒」「十分(じゅうぶん)」などの紛らわしい語は変換しない
英訳を表示確定した字幕の下に英訳を併記する（詳細）
高精度モード(fp32)認識精度がわずかに上がるが遅くなる。通常はオフでOK
ヒント スライダー系の設定は次回 ▶開始 時に反映されます。認識中に変えたら、いったん停止→開始してください。

3OBS に字幕を出す

1
OBS で「ソース追加」→「ブラウザ」を選ぶ
2
URL に http://localhost:8765 を入力（コックピット右下の「コピー」ボタンでも取得可）
3
幅・高さを配信解像度と同じにする（例: 1920 × 1080）

アプリ側で ▶開始 すると、喋った内容が OBS 側にも表示されます。スタイルやレイアウトの変更も即座に反映されます。

表示の流れ 話している間は薄い文字で途中経過が出て、ひと呼吸おくと確定文字に変わりエフェクトが発動します。

4単語スタジオ（誤認識対策とエフェクト）

コックピットの「✦ 単語スタジオ」から開きます。タブが4つあります。

🎤 ホットワード — 誤認識を減らす

自分の名前・チャンネル名・ゲームタイトルなど、よく誤認識される固有名詞を登録すると認識が改善します。

  • 表記 … 画面に出したい文字（例: 星野ひかり）
  • 読み … ひらがなの読み（例: ほしのひかり）。漢字を含む語は必ず入れる
  • スコア … 効きの強さ（省略可。目安 2.0〜3.0）
注意 ホットワードの変更は次回 ▶開始 時に反映されます。

✨ エフェクト単語 — 画面を彩る

特定の単語が字幕に出たとき、色・サイズ・アニメ・パーティクルで装飾します。 「ありがとう」「かわいい」「最高」など、よく使う言葉が最初から登録済みです。 行の右端のプレビューをクリックすると動きを確認できます。こちらは保存すると即反映です。

📁 単語プロファイル — 配信の種類で単語セットを切り替える

雑談用・ゲーム実況用・歌枠用など、配信の性質ごとの単語セットを作れます。 単語スタジオ上部の「編集する単語セット」で対象を選んで編集し、 配信時はコックピット左側の「単語プロファイル」で切り替えます。

  • プロファイルは「共通」に加算されます（置き換えではありません）。 名前・チャンネル名など常に使う単語は「共通」に入れておけば、どのプロファイルでも有効です
  • 同じ単語を両方に登録した場合はプロファイル側が優先されます
  • ホットワード・エフェクト単語・禁止ワード・英訳辞書の4種類すべてがプロファイル対象です

5スタイルスタジオ（見た目のカスタマイズ）

コックピット右側のプリセット一覧の ✎ ボタンから開きます。

文字スタイル（プリセット）

  • フォント・サイズ・色・フチ・グロー・登場アニメなどを調整
  • 「複製」で自分用プリセットを作るのがおすすめ
  • 途中表示の濃さ … 薄文字の透明度
  • 途中表示（薄文字）にもエフェクトを乗せる … 既定ではオフ（確定した瞬間にだけエフェクトが出る、落ち着いた見た目）

レイアウトボックス

  • 字幕の表示位置・サイズ・背景・行数などを定義
  • 通常の「流れる字幕」のほか、縦書き、画面に文字が散る「リリック」モードもあります

エクスポート / インポート（共有）

スタイルスタジオ下部の「📤 エクスポート」で、文字スタイルやボックスを .mojipack ファイルに書き出せます（保存先: data\export、 ボタンからフォルダを開けます）。もらったファイルは「📥 インポート」で取り込みます。 既存の定義は上書きされず追加され、同じ名前は「〜 (imported)」になります。

6英訳の併記

コックピットの「英訳を表示（各行の下）」をオンにすると、確定した字幕の下に英訳が小さく併記されます。 海外リスナー向けの配信にどうぞ。

  • 翻訳もローカルで動きます（ネット不要・追加料金なし）
  • 日本語の字幕が先に出て、英訳がひと呼吸おいて追いつきます
  • オンにして最初の ▶開始 時だけ「翻訳モデルをロード中...」と数秒待ちが入ります
固有名詞は「英訳辞書」で固定 人名・チャンネル名などは、単語スタジオの「🌐 英訳辞書」タブで 「字幕の表記 → 英訳」を登録すると、その語の訳を固定できます （例: 星野ひかり → Hikari Hoshino）。変更は次回開始時に反映されます。
品質について 軽量な翻訳モデルのため、辞書に無い固有名詞はうまく訳せないことがあります。 気になった誤訳があればぜひフィードバックしてください。

7フォルダとファイルの説明

ファイル / フォルダ内容
Mojicast.exeアプリ本体
_internal\実行に必要なファイル（触らない）
models\AIモデル一式（触らない）
data\あなたの設定・単語帳・スタイル一式。初回起動時に自動生成。 バックアップはこのフォルダをコピーするだけ
data\profiles\単語プロファイル（雑談用・ゲーム用など）。単語スタジオから作成
ui\ / overlay.html画面ファイル（触らない）

アップデート方法

1
Mojicast を終了する
2
旧フォルダ内の _internal フォルダを削除する
3
新しいZipを同じフォルダに上書き展開する

設定・単語帳・スタイル・AIモデル（models）はそのまま引き継がれます。 新しいZipにはデータファイルが含まれないため、あなたのカスタマイズは上書きされません。

アンインストールはフォルダごと削除するだけです。レジストリ等は使いません。

8困ったときは

症状対処
画面が真っ白「Microsoft Edge WebView2 ランタイム」をインストール（Web検索で公式Microsoftサイトから）。Windows 11 では通常発生しません
「ポート ○○ は使用中」アプリが既に起動しています。タスクトレイやタスクマネージャーで確認して二重起動を解消。 別アプリとの衝突ならコックピットの「OBS連携 → ポート」で変更（次回起動時反映・OBS側のURLも更新）
マイクを拾わないWindows 設定 → プライバシー → マイクの許可を確認。他アプリの占有も確認。マイク選択を「既定のマイク」に戻すのも有効
認識が変・遅い「無音で確定」「途中更新間隔」を調整。固有名詞の誤認識はホットワード登録で改善
OBS に何も出ないアプリが起動しているか、URL が http://localhost:8765 か、▶開始 しているかを確認。ブラウザソースを右クリック→「再読み込み」も有効
起動時に .NET / clr / Python.Runtime のエラー通常は起動時に自動解除されますが、まれに（読み取り専用の場所などで）残ることがあります。その場合はフォルダ内の ブロック解除.bat を一度実行してください
起動しない・すぐ落ちるフォルダをローカルディスクに置いているか確認（Zip内・ネットワークドライブは不可）。ウイルス対策ソフトの誤検知の場合は除外設定へ
