========================================
 Mojicast  —  テスター向け手順
========================================

配信用リアルタイム字幕アプリ（ローカル完結・オフライン動作）。
Python のインストールは不要。ネット接続は初回のAIモデルDLのときだけ必要で、以降はオフラインで動きます。
音声認識（日本語＋多言語）・句読点・翻訳（英中日韓ほか）・1対1コラボまで、すべてPCの中だけで処理します。
画面はライト基調です（ヘッダーの 🌙/☀ でダークにも切替できます）。

■ ご利用にあたって（免責）
  ・認識・翻訳・伏せ字は完璧ではなく、不適切な表示が出ることがあります。
    大事な場面はご自身でも確認してください。
  ・1対1コラボは「相手の声」を字幕化・ログ保存します。録音・プライバシーに
    関わるため、相手の同意を得たうえでご利用ください。
  ・本アプリは無保証（現状有姿・自己責任）です。詳しい免責は README を参照。

■ 動作環境
  - OS: Windows 10 / 11（64bit）
  - CPU: 最低 4コア8スレッド（2017年以降の Core i5 / Ryzen 5 相当）
         推奨 6コア以上（ゲーム配信と併用する場合）
  - メモリ: 最低 8GB / 推奨 16GB（英訳ON時、本アプリが約2GB使用）
  - ストレージ: 空き5GB（SSD推奨）
  - マイク / 初回のみネット接続（モデル約1.2GBのダウンロード）
  - WebView2 ランタイム（Windows 11 は標準搭載。Windows 10 で
    画面が真っ白なら「Microsoft Edge WebView2 Runtime」を入れてください）
  - OBS併用時はGPUエンコード（NVENC等）推奨

■ 初回起動について
  ・初回だけ「WindowsによってPCが保護されました」(SmartScreen)が出ることがあります。
    未署名アプリのためで、「詳細情報」→「実行」で起動できます（安全です）。
  ・ダウンロード由来のブロック(MOTW)は、アプリが起動時に自動で解除します。
    手動のブロック解除は不要です。
    （まれに自動解除できない場合のみ、フォルダ内「ブロック解除.bat」を一度実行）

■ 使い方
  1. このフォルダを丸ごとローカルディスクにコピー
     （ネットワークドライブやZip内から直接起動しないこと）
  2. Mojicast.exe をダブルクリック
  3. 初回はモデル読み込みで少し待ちます
  4. マイクを選んで ▶開始

■ どれを読めばいい？（逆引き）
  ブラウザで開ける説明書を同梱しています。ふだんはマニュアルだけでOK。
    マニュアル.html                … 操作全般（OBS連携・単語登録・翻訳・コラボ・トラブル対処）
    ガイド\スタイルガイド.html     … 字幕デザインの作り方（フォント・色・レイアウトのコツ、実例つき）
    ガイド\エフェクトガイド.html   … 「ありがとう」でキラキラ等、単語を光らせる・飛ばす演出の作り方
    ガイド\テクニカルガイド.html   … 詳しい人向け（CPU適性・搭載AIモデル・実測データ）

■ アップデート（旧バージョンからの上書き）
  1. Mojicast を終了する
  2. 旧フォルダ内の「_internal」フォルダを削除する（古い実行ファイルの残骸を避けるため）
  3. 新しいZipを同じフォルダに上書き展開する
  → 設定・単語帳・スタイル（dataフォルダ）・AIモデル(models)はそのまま引き継がれます。
     （新しいZipにはデータファイルが入っていないので、あなたの設定は上書きされません）
     ※旧バージョンでフォルダ直下にあったデータファイルは、初回起動時に自動で
       data フォルダへ移動します。
     ※アップデートで追加された新しい既定スタイルは、初回起動時に自動で一覧へ追加されます。

■ OBS 連携
  1. OBS でソース追加 →「ブラウザ」
  2. URL に  http://localhost:8765  （アプリ右下からコピー可）
  3. 幅・高さは配信解像度に合わせる（例 1920 x 1080）

■ フォルダの中身
  Mojicast.exe … 本体
  _internal\           … 実行に必要なファイル（触らない）
  models\              … AIモデル（触らない）
  ui\ / overlay.html   … 画面ファイル
  data\                … あなたの設定・単語帳・スタイル一式（初回起動時に自動生成）
                         バックアップは data フォルダをコピーするだけでOK

■ うまく動かないとき
  - 画面が真っ白 → WebView2 ランタイムを入れる
  - 「ポート ○○ は使用中」→ 既に起動中。二重起動していないか確認。
    別アプリと衝突する場合はコックピットの「OBS連携 → ポート」で変更できます
  - マイクが出ない → OSのマイク権限、他アプリの占有を確認
  - 「句読点／翻訳の読み込みに失敗」と出る → logs\load_error.log を添えて
    報告してください（原因調査に使います）
  - 気づいたことは何でもフィードバックください（誤認識の例・落ちた操作など）
